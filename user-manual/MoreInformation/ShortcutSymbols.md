# Keyboard Shortcut Symbols

Here is a list of the symbols used in this manual and in the app and the modifier key they represent.

| Modifier key | Symbol |
| ------------ | ------ |
| Command      | ⌘      |
| Shift        | ⇧      |
| Option       | ⌥      |
| Control      | ⌃      |
| Function     | fn     |
